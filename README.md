# SmartImage Pro - 智能图片批量处理工具

一键批量处理图片，无需安装配置，打开浏览器就能用！

## 功能

- **批量压缩** - 保持清晰度，文件减小 80%
- **批量缩放** - 统一尺寸，适配多平台
- **格式转换** - JPG/PNG/WebP 互转
- **灰度处理** - 一键去色

## 使用

```bash
npm install
npm start
# 打开 http://localhost:3000
```

Windows 用户双击 `start.bat` 即可。

## 技术栈

- Node.js + Express
- Sharp (图片处理)
- 纯前端 HTML/CSS/JS
